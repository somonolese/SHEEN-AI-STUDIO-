# ❄️ SHEEN

**SHEEN** is a modern, open-source Android app store inspired by the beauty and simplicity of Kashmir.

Designed with **Material 3** and **Material You**, SHEEN aims to provide a fast, elegant, and privacy-friendly experience for discovering, installing, and managing free and open-source Android applications.

> Built with ❤️ in Kashmir.

---

## ✨ Features

- 🏔️ Beautiful Material 3 interface
- 🎨 Material You dynamic colors
- 🧺 Basket for saving apps
- 📂 Categories & Collections
- 🔍 Powerful search
- 📥 Download Manager
- 🔄 Update Manager
- 📱 Android & Shizuku installer support
- 🗑️ Delete APK after installation
- 👤 Optional local profile picture
- 🌍 Multi-language support
- ⭐ Featured apps
- ❤️ Support SHEEN
- ⚡ Smooth animations
